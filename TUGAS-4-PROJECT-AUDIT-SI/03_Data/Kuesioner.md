# Kuesioner Audit Sistem Informasi
## PT Aegisindo Mitra Sejati — Framework COBIT 2019

---

## Petunjuk Pengisian

Mohon Bapak/Ibu memberikan penilaian terhadap setiap pernyataan di bawah ini dengan membubuhkan tanda centang (✓) pada salah satu kolom skala yang paling sesuai dengan kondisi nyata di PT Aegisindo Mitra Sejati saat ini.

| Skala | Keterangan | Nilai |
|-------|-----------|-------|
| **STS** | Sangat Tidak Setuju | 1 |
| **TS** | Tidak Setuju | 2 |
| **R** | Ragu-ragu / Netral | 3 |
| **S** | Setuju | 4 |
| **SS** | Sangat Setuju | 5 |

**Responden:** Karyawan PT Aegisindo Mitra Sejati yang terlibat langsung dalam pengelolaan data, aset, operasional, dan layanan TI perusahaan.

## Domain APO14 — Managed Data (Pengelolaan Data)

### Level 1 — Performed Process

| No | Pernyataan | STS (1) | TS (2) | R (3) | S (4) | SS (5) |
|----|-----------|---------|--------|--------|--------|---------|
| 1 | Perusahaan telah mendata dan mengelola data inventori, pelanggan, dan penjualan yang dibutuhkan untuk operasional sehari-hari | | | | | |
| 2 | Data produk dan stok dicatat dan diperbarui pada saat terjadi transaksi pembelian maupun penjualan | | | | | |
| 3 | Perusahaan menyimpan data pelanggan dan riwayat transaksi pada media penyimpanan tertentu (spreadsheet, aplikasi, atau marketplace) | | | | | |

### Level 2 — Managed Process

| No | Pernyataan | STS (1) | TS (2) | R (3) | S (4) | SS (5) |
|----|-----------|---------|--------|--------|--------|---------|
| 4 | Terdapat perencanaan yang jelas mengenai data apa saja yang harus dikelola dan siapa yang bertanggung jawab | | | | | |
| 5 | Kualitas dan konsistensi data antar saluran penjualan (marketplace, WhatsApp, toko) dipantau secara berkala | | | | | |
| 6 | Hasil pengelolaan data dievaluasi untuk memastikan data tetap akurat dan tidak duplikat | | | | | |

### Level 3 — Established Process

| No | Pernyataan | STS (1) | TS (2) | R (3) | S (4) | SS (5) |
|----|-----------|---------|--------|--------|--------|---------|
| 7 | Perusahaan memiliki prosedur standar terdokumentasi untuk pengelolaan dan pemutakhiran data | | | | | |
| 8 | Terdapat kebijakan klasifikasi dan kepemilikan data (data ownership) yang diterapkan secara konsisten | | | | | |
| 9 | Terdapat mekanisme integrasi data antar saluran penjualan sehingga data tersentralisasi dan tidak terpisah-pisah | | | | | |

## RACI Matrix

| Aktivitas | Manajer Operasional | Procurement & Gudang | Penjualan Online | IT / Admin Sistem | Keuangan & Adm | Customer Service |
|-----------|:-------------------:|:--------------------:|:----------------:|:-----------------:|:--------------:|:----------------:|
| Pengelolaan Data (APO14) | **A** | R | R | R | C | I |
| Pengelolaan Aset/Stok (BAI09) | **A** | **R** | C | R | I | I |
| Operasional Harian (DSS01) | **A** | R | **R** | C | I | R |
| Keberlangsungan Bisnis (DSS04) | **A** | C | I | **R** | C | I |
| Layanan Keamanan (DSS05) | **A** | I | C | **R** | C | I |

**Keterangan:**
- **R** = Responsible (melaksanakan tugas)
- **A** = Accountable (bertanggung jawab atas keputusan)
- **C** = Consulted (dimintai pendapat)
- **I** = Informed (diberikan informasi)
